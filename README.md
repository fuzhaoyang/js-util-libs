# js-util-libs(函数库)

## 概述

js 常见函数工具库

## Install(安装)

npm install js-util-libs --save

## Usage(使用)

1、防抖节流  
2、深拷贝  
3、call,apply,bind  
4、时间戳转时间格式  
5、转义字符转换  
6、匹配字符串中所有图片src  
7、滚动条滚动画缓冲


## 欢迎大家提PR扩充函数库，为开源社区贡献自己一份力
提Pr步骤  
1、src底下创建自己模块函数的文件夹  
2、函数模块包含markdow说明，有自己测试用例（必须）  
3、根部index.js导出函数  
4、packjson keywords里写自己函数关键字  
5、npm test 跑测试用例  
6、不要修改packjson版本号，版本号为线上最新用户使用版本  
7、Pull requests 测试用例过后，静等作者合代码
